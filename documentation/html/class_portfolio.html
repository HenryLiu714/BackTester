Detailed Description

The portfolio management system should keep track of all positions in a given portfolio, as well as generate orders of a quantity of stock based on signals.

Improvements include risk management and position sizing tools A portfolio object must be able to

  1. Handle SignalEvent objects
  2. Generate OrderEvent objects
  3. Interpret FillEvent objects to update positions
